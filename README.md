A clean and minimal **thinkboard / notes application** built using the MERN stack, designed for organizing ideas, thoughts, and short-form notes with a simple and intuitive user experience.

---

## Why this project?

Most notes apps either become bloated or overly simplistic.  
This project focuses on **clarity, responsiveness, and clean data handling**, while serving as a solid full-stack reference implementation using the MERN ecosystem.

It was built to practice:
- End-to-end full-stack development
- RESTful API design
- State management and persistence
- Clean UI + backend separation

---

## Key Features

- Create, edit, and delete notes
- Persistent storage using MongoDB
- RESTful backend API
- Responsive and minimal UI
- Clear separation between frontend and backend

## Getting Started

### Prerequisites
- Node.js
- MongoDB (local or cloud)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/MERN-THINKBOARD-NOTES-APP.git

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install


Running the App
# Start backend
cd server
npm start

# Start frontend
cd ../client
npm start
